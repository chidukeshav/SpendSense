# SpendSense 💰

SpendSense is an Android application that helps users track and manage their daily expenses easily.

## Features

* Add daily expenses
* View expense history
* Simple and user-friendly interface
* Helps users manage their budget

## Technologies Used

* Android Studio
* Kotlin / Java
* XML (UI Design)
* Gradle

## How to Run the Project

1. Clone the repository
2. Open the project in Android Studio
3. Sync Gradle
4. Run the app on emulator or Android device
